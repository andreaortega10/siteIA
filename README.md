# WebGen AI – Protótipo 

## Descrição

O WebGen AI é um protótipo de interface web que simula uma aplicação de inteligência artificial voltada à geração de ideias e estruturas iniciais para sistemas web a partir de linguagem natural.

O sistema permite que o usuário interaja com uma IA simulada por meio de um chat, recebendo respostas sobre:
- conceitos básicos de tecnologia
- funcionamento da plataforma
- orientações iniciais para criação de sistemas

## Objetivo

Demonstrar, em nível de front-end, a experiência de uso de uma ferramenta de IA aplicada à construção de sistemas web, sem integração com APIs externas.

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (puro)  
- Bootstrap 5 (CDN)  
- Bootstrap Icons  

## Funcionalidades

- Interface moderna em tema escuro com alternância para modo claro
- Campo de entrada centralizado para interação com o usuário
- Simulação de respostas de IA com base em palavras-chave
- Chat dinâmico com exibição de mensagens
- Validação de entrada (mensagem vazia)
- Toasts de aviso
- Modal de saída (logout simulado)
- Persistência de tema (localStorage)

## Comportamento da IA (simulada)

A IA responde de forma contextual a perguntas como:
- introdução ao sistema
- o que é inteligência artificial
- conceitos básicos de HTML, CSS e JavaScript
- como iniciar um projeto

Quando não reconhece a pergunta, informa que ainda está em desenvolvimento.
